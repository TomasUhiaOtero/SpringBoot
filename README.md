# Spring Boot Projects

Este repositorio contiene múltiples proyectos de Spring Boot organizados en carpetas separadas.

## Estructura del Repositorio

Cada carpeta en este repositorio representa un proyecto independiente de Spring Boot:

- `springboot-applications/` - [Descripción del proyecto]

## Cómo Usar

Cada proyecto tiene su propio README con instrucciones específicas de configuración y ejecución.

Para trabajar con un proyecto específico:

```bash
cd <nombre-del-proyecto>
# Seguir las instrucciones del README del proyecto
```
